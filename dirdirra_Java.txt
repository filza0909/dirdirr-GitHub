*Figma adalah sebuah aplikasi desain berbasis cloud yang banyak digunakan untuk membuat tampilan antarmuka (UI) dan pengalaman pengguna (UX).
*Aplikasi berbasis Vektor(elemen yg didalam tidak akan pecah karna bentuk garis)
*Keunggulan Figma:
1.Berbasis cloud,yang langsung tersimpan di server
2.Dapat berkolaborasi real time,dan bisa bekerja sama dalam file yang sama
3.Prototyping → membuat desain interaktif dan bisa diuji langsung.
4.Popularitas → banyak dipakai perusahaan untuk mendesain produk digital karena praktis, fleksibel, dan mendukung kerja tim jarak jauh (remote).

*Perbandingan: lebih ringan daripada Adobe XD dan lebih fleksibel daripada Sketch karena berbasis cloud.
*Komponen utama interface Figma:
1.Layar panel → menampilkan struktur layer.
2.Toolbar → berisi tools utama.
3.Kanvas → area kerja utama untuk mendesain.
4.Property panel → tempat mengatur detail objek/desain
